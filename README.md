# Automation Testing Task

Project Overview:
This project contains automated test scripts developed using Selenium WebDriver with Java.
The scripts automate core functionalities such as login, search, and navigation.

Tools & Technologies Used:
- Java
- Selenium WebDriver
- Maven
- Chrome Browser
- Eclipse IDE

Project Structure:
- src/test/java → Contains automation test scripts
- pom.xml → Contains required dependencies

How to Run the Project:
1. Download or clone the project.
2. Open the project in Eclipse IDE.
3. Right click on the project → Maven → Update Project.
4. Navigate to src/test/java.
5. Open the main class file.
6. Right click → Run As → Java Application.

Test Scenarios Automated:
- Verify login functionality
- Verify search functionality
- Verify product selection
- Verify logout functionality
